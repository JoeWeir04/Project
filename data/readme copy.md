# Data Folder
## /graphs
This folder contains graphs created from scripts in the data parent folder. These scripts are `get_nasa_tlx_plot.py`, `Distance_vs_Angle_plot.py`, `violin_plot.py`. All R notebooks in the notebooks folder were also used to create graphs in this folder.

## /transcripts

This folder contains all coded transcripts from participants as word documents in the naming convention `Interview [number].docx`. It also contains a word document of all codes - `Codes.docx`, and an excel sheet of all coded exerpts `Codes.xlsx`.

## /raw 
This folder contains data that was collected through loggin on the meta quest 3 headset, these files are `VR_log.csv` and `VR_paths.csv`. This folder also contains data from microphone array benchmarking, These files are the csv files named with `angles[number].csv` and the excel file `Benchmark.xlsx`.

This folder also contains the raw exported data from the qualtrics surveys these files are `PerVisualisationRaw.csv` and `AllVisualisastions.csv`.
## /surveys 
This folder contains pdf copies of the surveys used to gather responses from participants.
## /tables 
This folder cantains tables produced by scripts found in the parent data folder. The scripts used to produce these tables are `get_nasa_tlx_plot.py` and `violin_plot.py`.
## /processed
This folder contains all processed data produced from the scripts found in thee parent data folder. The scripts used were `Clean_all_visualisations_survey_data.py`, `Clean_log_data.py` and `Clean_condition_survey_data.py`.
## /notebooks
This folder contains R notebooks which were used to produce figures and plots from the data in the processed folder.
## /ethics
This folder contains the signed project ethics checklist. It also contains the introduction and debrief scripts for the experiment.
